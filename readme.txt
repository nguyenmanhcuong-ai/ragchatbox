*** RUN THE APPLICATION ***
1. Activate the virtual environment:
	source venv/bin/activate # Linux/Mac
	venv\Scripts\activate # Windows

2. Install the necessary libraries: Copy the libraries in the setup.txt file

3. Run the application: streamlit run main.py

4. Access the application at: http://localhost:8501


*** SYSTEM INTRODUCTION ***
Smart assistant application supports looking up and consulting on Vietnam's Road Traffic Law, using AI technology to analyze and answer questions accurately and easily.


*** MAIN FEATURES ***
1. Automatically process and analyze legal documents
2. Smart answers with law citations
3. Search by topic and keyword
4. Save search history
5. Suggest questions by topic
6. Show source citation information
